Smart Home Hub

ESP32-based MQTT gateway for centralized device management.

Project Description

This project serves as the central brain for the local IoT network. It connects to the WiFi network and uses the PubSubClient library to communicate with a Mosquitto MQTT broker. It handles relay switching for high-voltage appliances and monitors ambient light levels using an LDR.

Components

  • ESP32 DevKit V1
  • 4-Channel 5V Relay Module
  • LDR (Light Dependent Resistor)
  • Hi-Link HLK-PM01 (5V Step-down)
  • Jumper Wires & Breadboard

Main Sketch (.ino)

// Example Snippet
void setup() {

  if (!client.connected()) reconnect();
  client.loop();
